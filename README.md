A smart, automated daily planner that adjusts your tasks and recommendations in real time according to the weather. Never miss an outdoor activity or get caught in the rain again — let your planner adapt and optimize your day, seamlessly!

## Features

- **Real-time Weather Integration**: Automatically pulls live weather data to adjust your daily itinerary.
- **Smart Scheduling**: Prioritizes and reschedules tasks based on weather conditions.
- **Customizable Preferences**: Choose which activities are weather-sensitive.
- **User-friendly Interface**: Easily add, remove, and modify tasks.
- **Mobile & Desktop Friendly**: Designed for all devices.

## Getting Started

1. **Clone the repository**
    ```bash
    git clone https://github.com/PreetKot/Automated-weather-based-daily-planner.git
    cd Automated-weather-based-daily-planner
    ```
2. **Install dependencies**  
   Make sure you have Python and Pip installed.
    ```bash
    pip install -r requirements.txt
    ```
3. **Set up your weather API key**
   - Sign up for a free API key from [OpenWeatherMap](https://openweathermap.org/api) or your chosen service.
   - Enter your API key in the relevant config file or environment variable.

4. **Run the planner**
    ```bash
    python main.py
    ```

## Usage

- Add your daily tasks through the UI
- The planner will automatically suggest optimal scheduling
- Tasks that depend on weather conditions (e.g. jogging, picnics) are highlighted and adjusted automatically if weather changes.
